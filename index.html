História dos computadores

A Origem O Desenvolvimento computacional remonta a idades mais remotas se formos nos pegar a história.
Mark I, foi o primeiro computador moderno, dando inicio a computação contemporanêa.
Em seguida veio a chamada Arpanet que foi o primeiro método utilizado para comunicação entre computadores.
Tim Bernes-lee cunhou e criou a chamda World Wide Web (www) tornando assim a internet mais acessível.

O que são Clients?

Podemos definir Clients, ou clientes em português, como aquele que acessa a informação
Ou seja, qualquer pessoa do mundo que tenha um computador pessoal, celular, tablet ou até mesmo um televisão.

Resumindo,Clients são os usuários que acessam sistemas, sites na web ou qualquer outro programa e aplicativos.

O que são Servers?

Servers, ou servidores, são locais onde os clients acessarão a informação que estão procurando. Esse acesso não necessaraimente é feito de forma direta. Por exemplo, um site como o da DIO, está localizado em algum servidor. Nós alunos iremos acessar o site da DIO pela nossa máquina que irá se conectar com o servidor de hospedagem da dio que nos mostrará este mesmo site.

Como é uma estrutura básica de um HTML?

A estrutura mais básica de um HTML pode ser definida pelos usos das tags "html", "head", "body", de maneira encadeadas. Existem inúmeras tags em uma página HTML, colocaremos aqui as principais usadas:
tags: a, img, h1 a h6, p, strong, u, i.
Ainda, para cada tag podem existem atributos como: href para a tag "a", ou src para a tag "img"

Entendendo a diferença entre listas ordenadas e não ordenadas

Uma lista ordenada é aquela que é numerada em ordem crescente como o exemplo abaixo:

  1. Item
  2. Item 2
  3. Item 3
Por outro lado uma lista não ordenada não contem numeração, veja abaixo:
  • Item 1
  • Item 2
  • Item 3

Ainda, podemos fazer uma lista não-ordenada dentro de outra:

  • Item 1
  • Item 2
  • Item 3
    • Item 3.1
    • Item 3.2
    • Item 3.3
